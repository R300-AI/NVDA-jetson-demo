建立一個簡單的 CNN 模型，檢視模型結構與參數數量
---

**題目說明**
1. 以 **PyTorch** 定義 `SmallCNN`：輸入 `N × 3 × 32 × 32`，輸出 10 類。模型路徑：`conv1 → ReLU → MaxPool → conv2 → ReLU → Flatten → FC`。
2. 使用**合成資料**或 **CIFAR‑10**（二選一）進行一次 forward 驗證。
3. 列印：模型結構、參數總數與可訓練參數、逐層參數形狀。
4. 若 GPU 可用，自動切換 **CUDA**；forward 完成後在 CUDA 上進行同步以確保量測時間正確。
